## How we work

1. **매주 하나의 Tale.** 예외 없이.
2. **말이 아닌 증명.** 코드와 사용자 피드백으로만 검증합니다.
3. **넓이보다 깊이.** 일주일 동안 도메인의 바닥까지 갑니다.
4. **미련 없는 종결.** 버려진 Tale이 다음 편의 연료입니다.
5. **승격은 의식.** 살아남은 Tale만이 Saga가 됩니다.

자세한 내용은 [MANIFESTO](./MANIFESTO.md)에서 읽을 수 있습니다.
